# Sales & Revenue Analysis Dashboard

This project is developed using Python, Pandas, and Streamlit.

## Features
- Upload CSV files.
- Analyze total sales and total revenue.
- Visualize sales data using charts.
- Display top-performing products.
- Interactive and user-friendly dashboard.

## Technologies Used
- Python
- Pandas
- Streamlit

## Project Purpose
This project was created as part of my Data Analytics Internship task to perform sales and revenue analysis using Python.
